Privacy Policy

Last updated: 5 June 2026

MIQ-Win is a free, open-source QuickLook plugin for Windows. This website and the plugin set no cookies and use no analytics or tracking. The only personal data involved is described below.

Website

This site is hosted on GitHub Pages. When you visit, GitHub Inc. (USA) automatically records technical data such as your IP address and browser type in its server logs, and the embedded screenshots are loaded from GitHub servers. This processing is carried out by GitHub under its privacy statement.

Plugin

The MIQ-Win plugin runs entirely on your PC, inside QuickLook. It does not transmit your files or any usage data, and it makes no network connections of its own. It has no telemetry and no built-in updater. Downloading the plugin and checking for new releases happen through your browser on GitHub; those requests reveal your IP address to GitHub, as with any web request.

Your rights

Under the Swiss Federal Act on Data Protection (revFADP) you may request access to, correction of, or deletion of personal data concerning you. As this project holds no personal data of its own, such requests relate to the logs held by GitHub. For anything regarding this project, contact the operator below.
